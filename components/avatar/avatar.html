Avatar

Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.

Profile Avatar

Profile Avatars can be created using the class avatar, inside the img tag.

Sizes of Avatar

Size of the avatars can be changed by adding avatar--large avatar--xlarge avatar avatar--xxlarge class to the existing avatar class.

Square Avatar

Square avatars can be created by adding square class to the existing avatar class.

Text Avatar

Text avatars can be created by adding avatar--text class to the existing avatar class.
