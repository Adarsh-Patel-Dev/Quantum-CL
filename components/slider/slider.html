Slider

Slider is a input element of type "range". It defines a control for entering a number. It allows users to easily input values in range by sliding the slider. Default range is 0 to 100. However, you can set restrictions on what numbers are accepted with the following attributes.

max - specifies the maximum value allowed
min - specifies the minimum value allowed
step - specifies the legal number intervals
value - Specifies the default value
